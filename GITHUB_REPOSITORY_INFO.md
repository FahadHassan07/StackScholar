## 🚀 How to Apply These Settings

### 1. Repository Description
1. Go to your repository: https://github.com/MalithShehan/AI-Smart-Study-Assistant
2. Click **⚙️ Settings** (or **About** section on right side)
3. Paste the description
4. Click **Save changes**

### 2. Topics
1. On repository main page
2. Click **⚙️** gear icon next to "About"
3. Add topics (comma-separated or click suggestions)
4. Save

### 3. Social Preview
1. Go to **Settings** → **Social Preview**
2. Upload image (1280x640px)
3. Save

### 4. Enable Features
1. Go to **Settings** → **General**
2. Scroll to **Features**
3. Check boxes for Issues, Discussions, etc.
4. Save
